## 📝 技术说明

### 快捷键列表
| 快捷键 | 功能 |
|--------|------|
| Ctrl+N | 新建任务 |
| Ctrl+F | 聚焦搜索框 |
| Ctrl+S | 保存数据 |
| F5 | 刷新当前视图 |
| Ctrl+1 | 项目概览 |
| Ctrl+2 | 任务管理 |
| Ctrl+3 | 开发计划（里程碑） |
| Ctrl+4 | 活跃热力图 |
| Ctrl+5 | 资源浏览器 |
| Ctrl+6 | 日历视图 |
| Ctrl+7 | 工时周报 |
| Ctrl+8 | 版本对比 |
| Ctrl+9 | 发布管理 |

### CSV 导出
- 任务导出：标题、描述、分类、状态、紧急度、重要度、标签、里程碑、创建日期、完成日期、截止日期、预计工时、已记录工时
- 里程碑导出：标题、版本、描述、开始日期、目标日期、是否完成
